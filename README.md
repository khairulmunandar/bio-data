# 🌟 Khairul Munandar - Portfolio Website

![Portfolio Preview](https://placehold.co/800x400/0a0f1c/00d4ff?text=Khairul+Munandar+Portfolio)

Portfolio website modern untuk Khairul Munandar - Mahasiswa Teknik Informatika yang menampilkan pengalaman, keahlian, proyek software & hardware, dan galeri kegiatan.

## 🚀 Fitur Utama

- ✅ **Responsive Design** - Tampil sempurna di desktop, tablet, dan mobile
- ✅ **Hamburger Menu** - Navigasi yang nyaman di perangkat mobile
- ✅ **Animasi Modern** - Efek float, pulse, slide, dan scroll reveal
- ✅ **2 Kategori Proyek** - Software/Web Development & Hardware/Embedded Systems
- ✅ **Dark Theme** - Desain modern dengan warna gelap yang elegan
- ✅ **Kompatibel Lintas Platform** - Android, iOS, Windows, macOS

## 🛠️ Teknologi yang Digunakan

| Teknologi | Keterangan |
|-----------|------------|
| HTML5 | Struktur semantik modern |
| CSS3 | Flexbox, Grid, Animations, Variables |
| JavaScript | Hamburger menu, smooth scroll, Intersection Observer |
| Font Awesome 6 | Ikon profesional |
| Google Fonts | Plus Jakarta Sans |
